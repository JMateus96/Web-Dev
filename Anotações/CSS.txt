CSS

Dentro do próprio código de programação

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
            <meta charset="utf-8">
        
            <title>Título</title>
            <style type="text/css">

Por conta deste último parágrafo o programa de editor de código-fonte (Visual Code, neste caso) consegue identificar que a linguagem que será utilizada neste documento será css.

As regras do CSS são feitas em outra aba do código de programação

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
            <meta charset="utf-8">
        
            <title>Título</title>
            <link rel="stylesheet" href='audio.css'>

Por conta deste último parágrafo o programa de editor de código-fonte (Visual Code, neste caso) consegue buscar através da tag href que a linguagem que será utilizada neste documento serão aquelas contidas no link 'audio.css' que contém as regras que serão utilizadas no css

Styles in line
As regras do CSS são feitos dentro do próprio código de programação, porém não é tão recomendável por conta que pode acabar deixando muito grande os códigos.
